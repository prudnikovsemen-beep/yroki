# Виджет банковских операций

Виджет для отображения последних успешных банковских операций клиента с возможностью фильтрации и сортировки.

Проект представляет собой Python‑библиотеку для работы с банковскими операциями. Основные возможности:

* маскировка номеров банковских карт и счетов;
* фильтрация операций по статусу (выполненные, отменённые и т. д.);
* сортировка операций по дате (по возрастанию/убыванию);
* преобразование форматов даты;
* обработка данных в едином интерфейсе.

Описание компонентов
Корневые файлы
.gitignore — содержит шаблоны для игнорирования временных и системных файлов (кэши, IDE‑файлы, виртуальные окружения и т. д.).

pyproject.toml — главный конфигурационный файл проекта:

зависимости (основные и для разработки);

настройки инструментов линтинга (flake8, black, isort, mypy);

конфигурация pytest.

README.md — файл с описанием проекта (рекомендуется создать для документирования).

Директория src/
Содержит исходный код приложения:

__init__.py — делает директорию src пакетом Python.

masks.py — реализует базовые функции маскировки:

get_mask_card_number(card_number: str) -> str — маскирует номер карты в формате XXXX XX** **** XXXX;

get_mask_account(account_number: str) -> str — маскирует номер счёта в формате **XXXX.

widget.py — новый модуль с расширенными функциями:

mask_account_card(input_string: str) -> str — обрабатывает строки вида "Visa Platinum 7000792289606361" или "Счёт 73654108430135874305", вызывает соответствующие функции маскировки из masks.py;

get_date(date_string: str) -> str — преобразует дату из формата ISO ("2024-03-11T02:26:18.671407") в формат "ДД.ММ.ГГГГ" ("11.03.2024").

Директория tests/
Содержит тесты для проверки корректности работы кода:

__init__.py — инициализирует директорию как пакет Python.

tests_masks.py — тесты для функций из masks.py:

проверка корректной маскировки карт и счетов;

обработка номеров с пробелами;

валидация длины и типа данных (ошибки для неверных входных данных).

test_widget.py — тесты для функций из widget.py:

test_mask_account_card_card() — тест маскировки карт через mask_account_card;

test_mask_account_card_account() — тест маскировки счетов через mask_account_card;

test_get_date_valid() — проверка преобразования даты в нужный формат;

test_get_date_edge_cases() — тесты крайних случаев (например, 01.01.2000 и 31.12.2099).

Директория .git/
Создаётся автоматически при выполнении git init. Содержит:

историю коммитов;

конфигурацию репозитория;

данные о ветках и т. д.
